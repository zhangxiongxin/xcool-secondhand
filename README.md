# 校园二手物品交易
> 技术栈：vue+flask+mysql


## Features
1. 轻量、高效的构建前后端分离web应用；
2. 图片等静态资源存储于“七牛云”以加快访问速度；
3. 第三方短信验证接口以及第四方在线支付。


## Usage

### Start
```bash
# install vue-cli
npm install vue-cli -g

# install dependencies
npm install

# serve with hot reload
npm run dev
```


> **Remind**
- 建议在linux下运行


### Build
```
npm run build
```
